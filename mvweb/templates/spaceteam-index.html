How To Make Your Own Isk War
Step 1

Make a copy of this template Google Sheet

Step 2

Edit the metadata sheet to reflect the conflict you're describing

Step 3

Edit the events sheet to describe the dates each corp joined/left the war.

Alliance tags are optional - providing them with join events allows the compilation of a hierarchical list of war participants.

Theres's no need to provide alliance tags with leave events, it doesn't do anything

Step 4

Publish the sheet as CSV:

File menu -> Share -> Publish to web

Entire Document: Comma-separated values (.csv)

Press Publish and confirm

Step 5

Take the new link provided in the Publish window, and extract the ID from it. The ID is the long series of characters after /spreadsheets/d/e/ and before /pub?

For example, in the template URL:

https://docs.google.com/spreadsheets/d/e/2PACX-1vTGA3BrnYMt0_l7ZxvX2K6_zLZRiXK4nEYwuBxoQGHDaiQZcWqSJWO-JifVBRs0iWVtAZzMq2hUv3ku/pub?output=csv

The ID is the bold red section. Copy the ID

Step 6

Paste the ID after

https://marioview.honk.click/spaceteam/stats/

Now you have the URL to your new isk war website

Notes

There's a 10 minute cache timer on each sheet ID - if you make a change on the Google Sheet, it can take up to 10 minutes to reflect on the Spaceteam view.

If your sheet seems like it isn't working right, you can replace /stats/ with /check/ for a debug output.

If your sheet is REALLY broken, you'll get auto redirected to the check interface.

The spaceteam stats that this site provides are calculated solely from the echoes.mobi dataset and whichever corresponding team event sheet you're looking at. If you don't like it just make your own
